# Team Availability Tracker

## Description

The Team Availability Tracker is a Flask-based web application that displays a list of team members and their availability status. Users can change a member's status between **Available** and **Busy** using a toggle button. Every change is stored in a SQLite database and reflected immediately on the dashboard.

## Features

- View team members
- Toggle availability status
- SQLite database storage
- Dynamic status updates
- Responsive interface

## Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite

## Run the Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
